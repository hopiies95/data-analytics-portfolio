# SQL Business Analysis Project

## Project Overview
This project analyzes structured sales data to identify revenue trends, top customers, and product performance. The goal is to simulate real-world business reporting using SQL queries.

## Tools Used
- PostgreSQL
- SQL

## Skills Demonstrated
- GROUP BY and aggregation
- ORDER BY and sorting
- Data summarization
- Business performance analysis

## Business Questions Answered
1. Which customers generate the highest revenue?
2. Which products perform best?
3. What patterns can be observed in overall sales?

This project demonstrates my ability to query relational databases and transform raw data into actionable business insights.
